WindowList
===========

Mac OS X command line utility to list windows.

## Example output

    PID	  WindowID  Title
    68854	11301	    Security & Privacy
    69363	11606	    SourceTree
    69363	11605	    (null)
    69363	11604	    (null)
    70244	11790	    Ars Technica

## Usage

    ./WindowList

## Building

Run

    make
